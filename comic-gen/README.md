# comic-gen

生成四格漫画（起承转合结构）。用户提供故事大纲，自动生成完整的四格漫画。

## 特点

- **自动分镜**：根据故事大纲自动拆分为 4 个场景
- **对话优化**：自动精简对话，确保每个气泡不超过 20 字
- **风格一致**：所有角色在 Panel 1-3 保持一致（Panel 4 可以风格突变）
- **视觉元素**：自动添加表情、动作、符号（汗滴、问号、爆炸等）

## 使用方式

### 交互式创作（推荐）

直接告诉 AI 你的故事想法，AI 会帮你完善并生成漫画：

```
我想画一个程序员没有 AI 就下班的故事，最后反转到未来战争
```

### 脚本生成（开发中）

```bash
bash scripts/gen-comic.sh -o ./my-comic.png -s ./story.yaml
```

## 示例

- `examples/programmer-war.yaml` - 程序员危机：从办公室到战场

## 依赖

- `gemini-image-gen` skill
- `GEMINI_API_KEY` 环境变量
- SOCKS5 代理（国内必须）

## 适用场景

- 技术段子、程序员日常
- 概念解释、技术演进
- 产品吐槽、需求变更
- 未来想象、科技预测
