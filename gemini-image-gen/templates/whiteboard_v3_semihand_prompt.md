# Whiteboard Edit Prompt — v3 (SEMI-HANDDRAWN)

Use this as an **image-edit** prompt (photo → online clean diagram) while preserving the original semantics/layout.

```text
Redraw this photographed whiteboard into a clean ONLINE diagram with a SEMI-HANDDRAWN style.

PRESERVE SEMANTICS + LAYOUT: Keep the same structure, arrows, boxes, numbering, and all original text. Do NOT add or remove content.

SEMI-HANDDRAWN STYLE:
- Lines are mostly straight and aligned, but still look like marker strokes (subtle wobble).
- Text uses a handwritten look, but more consistent baseline/spacing than pure hand-drawn.
- Keep whitespace and original layout proportions.

CLEANUP: Remove glare/shadows, correct perspective, make everything crisp and readable.

COLORS: Blue marker lines (#1E5EFF), text dark gray (#222), white background.

OUTPUT: 4:3, 4K, like an exported online whiteboard (handwritten aesthetic).
```
